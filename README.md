Konfigurasi Kode LokalUnduh atau clone repositori ini.Buka file app.js.Di bagian paling atas, ganti nilai supabaseUrl dan supabaseAnonKey dengan kunci yang Anda dapatkan dari dasbor Supabase Anda.// app.js
